入驻流程
  • 注册 点击网站首页的“注册”按钮，进入注册界面，选择注册类型，根据提示完成注册
  • 店铺申请 注册成功后，进入个人主页里的“店铺申请”界面，根据提示提交相关证件及基本资料
  • 后台审核 店铺资料提交完成后，后台会在3个工作日内审核您的资料并通知您审核结果
  • 店铺开通 审核通过后，点击网站首页“我的翻译”进入个人主页，完善“店铺资料”“添加商品”，进行“银行卡管理”
